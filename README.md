# Carolina Cunha Levek | Site Institucional

Landing page em HTML puro para a psicóloga Carolina Cunha Levek, com CSS externo e interações em JavaScript sem bibliotecas.

## Estrutura

- `index.html`: página única do site
- `styles.css`: estilos visuais e responsividade

## Identidade do projeto

- Idioma: `pt-BR`
- Codificação: `UTF-8`
- Conceito visual: "Raízes e Vínculos"
- Tipografia: Playfair Display, DM Sans e Cormorant Garamond
