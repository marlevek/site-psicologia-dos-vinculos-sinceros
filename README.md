# Carolina Cunha Levek | Site Institucional

Landing page em HTML puro para a psicóloga Carolina Cunha Levek, com um único CSS externo e interações em JavaScript sem bibliotecas.

## Estrutura

- `index.html`: página única do site
- `styles-v2.css`: estilos visuais e responsividade
- `robots.txt`: indexação final para buscadores
- `sitemap.xml`: sitemap da home publicada

## Identidade do projeto

- Idioma: `pt-BR`
- Codificação: `UTF-8`
- Conceito visual: "Raízes e Vínculos"
- Tipografia: DM Sans e Cormorant Garamond

## Publicação

- Domínio final: `https://carolinacunhalevek.com.br/`
- Imagem principal otimizada: `foto_carol_psico.webp`
